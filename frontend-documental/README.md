# Arqui - Sistema de Gestión Documental

Sistema de gestión documental especializado para proyectos de arquitectura.

## Características

- Dashboard con estadísticas de documentos
- Filtros avanzados de búsqueda
- Tabla responsive de documentos
- Modal de detalles de documentos
- Tema de colores arquitectónico
- Diseño responsive

## Instalación

```bash
npm install
npm start
